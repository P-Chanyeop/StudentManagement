# 디자인 변경 계획서

## 📋 개요

rallyz.co.kr 스타일을 참고하여 학원 관리 시스템의 전체적인 UI/UX를 현대적이고 세련되게 개선합니다.
**네이버 그린 컬러 테마(#03C75A)는 유지**하면서 디자인 퀄리티를 대폭 향상시킵니다.

---

## 🎨 주요 변경 사항

### 1. **헤더 (Header)**

#### 기존 디자인:
- 단순한 흰색 배경
- 우측에만 사용자 정보
- 미니멀한 스타일

#### 새로운 디자인:
- **로고 추가**: 좌측에 K-Play Academy 브랜드 로고와 아이콘
- **네비게이션 메뉴**: 중앙에 대시보드, 학생 관리, 수업 관리 등 주요 메뉴
- **향상된 사용자 정보**: 아바타 + 이름 + 이메일 표시
- **Sticky Header**: 스크롤 시에도 상단 고정
- **Hover 효과**: 메뉴 아이템에 언더라인 애니메이션

```css
높이: 72px (기존 60px에서 증가)
그림자: 더 부드러운 box-shadow
네비게이션: active 상태 시 하단 녹색 바
```

---

### 2. **히어로 섹션 (Hero Section)** - 신규 추가

#### 새로운 기능:
- **그라데이션 배경**: 네이버 그린 계열의 아름다운 그라데이션
- **환영 메시지**: "안녕하세요, 관리자님! 👋"
- **서브타이틀**: 간단한 설명 문구
- **장식 요소**: 반투명 원형 배경으로 시각적 풍부함

```css
배경: linear-gradient(135deg, #03C75A 0%, #02B350 100%)
패딩: 64px (모바일: 48px)
텍스트 색상: 흰색
```

---

### 3. **통계 카드 (Stats Cards)**

#### 기존 디자인:
- 아이콘 + 레이블 + 값
- 간단한 hover 효과
- 파스텔 컬러 아이콘 배경

#### 새로운 디자인:
- **상단 바 애니메이션**: hover 시 녹색 바가 위에서 나타남
- **그라데이션 아이콘**: 네이버 그린 그라데이션 + 그림자 효과
- **트렌드 배지**: 우측 상단에 증감률 표시 (+12%, +8% 등)
- **푸터 정보**: 하단에 추가 정보 (예: "지난달 대비 12% 증가")
- **더 큰 값**: 36px 폰트 크기로 가독성 향상
- **향상된 그림자**: hover 시 녹색 그림자 효과

```css
border-radius: 16px (기존 12px)
아이콘 크기: 56px
값 폰트 크기: 36px (기존 28px)
hover 시 transform: translateY(-4px)
box-shadow: 0 12px 24px rgba(3, 199, 90, 0.15)
```

---

### 4. **대시보드 카드 (Dashboard Cards)**

#### 기존 디자인:
- 흰색 배경 카드
- 간단한 헤더
- 리스트 아이템

#### 새로운 디자인:
- **헤더 배경**: 연한 회색 배경으로 구분 명확화
- **아이콘이 있는 타이틀**: 각 카드 제목에 관련 아이콘 추가
- **컬러풀한 배지**: 카드 우측에 개수/상태 표시
- **아이템 아이콘**: 각 리스트 아이템에 둥근 배경의 아이콘
- **더 큰 padding**: 24px → 28px로 여유로운 공간
- **hover 효과**: 카드 전체와 아이템별 hover 효과

```css
card-header 배경: var(--gray-50)
list-item padding: 20px 28px (기존 16px 24px)
item-icon: 44px 둥근 사각형 배경
```

---

### 5. **배지 및 상태 표시**

#### 새로운 디자인:
- **더 큰 배지**: padding 증가, 더 눈에 띄는 크기
- **둥근 모서리**: border-radius 12px-14px
- **의미 있는 색상**:
  - 성공/출석/확정: 녹색 계열
  - 경고/지각/대기: 노란색 계열
  - 위험/결석/취소: 빨간색 계열
  - 정보/진행중: 파란색 계열

---

### 6. **기능 카드 그리드 (Feature Cards)** - 신규 추가

#### 새로운 기능:
- **6개 주요 기능 카드**: 학생 관리, 수업 관리, 출결 관리, 수강권 관리, 상담 관리, 통계 분석
- **큰 아이콘**: 72px 크기의 아이콘
- **그라데이션 배경**: 아이콘 배경은 연한 녹색 그라데이션
- **큰 hover 효과**: translateY(-8px)로 크게 뜨는 효과
- **설명 텍스트**: 각 기능에 대한 간단한 설명

```css
grid: repeat(auto-fill, minmax(320px, 1fr))
padding: 32px
icon 크기: 72px
hover transform: translateY(-8px)
```

---

### 7. **CTA 섹션 (Call-to-Action)** - 신규 추가

#### 새로운 기능:
- **눈에 띄는 그라데이션 배경**: 네이버 그린
- **큰 제목**: "지금 바로 시작하세요"
- **행동 유도 버튼**: 흰색 버튼 + 녹색 텍스트
- **장식 요소**: 반투명 원형 배경

---

### 8. **푸터 (Footer)** - 신규 추가

#### 새로운 기능:
- **어두운 배경**: gray-900
- **회사 정보**: 저작권, 연락처 정보
- **중앙 정렬**: 깔끔한 레이아웃

## 🎯 디자인 원칙

### 1. **일관성 (Consistency)**
- 모든 페이지에서 동일한 컴포넌트 스타일 사용
- 통일된 border-radius (12px, 14px, 16px, 18px 등)
- 일관된 spacing 시스템 (8px 단위)

### 2. **계층 구조 (Hierarchy)**
- 명확한 시각적 계층 구조
- 중요한 정보는 크고 굵게
- 덜 중요한 정보는 작고 연하게

### 3. **피드백 (Feedback)**
- 모든 인터랙티브 요소에 hover 효과
- transition 애니메이션 (0.2s ~ 0.3s)
- transform 효과로 입체감 제공

### 4. **가독성 (Readability)**
- 충분한 padding과 margin
- 적절한 폰트 크기 (최소 13px)
- 명확한 색상 대비

### 5. **반응형 (Responsive)**
- 모바일, 태블릿, 데스크톱 모두 지원
- 유동적인 grid 레이아웃
- 작은 화면에서는 단순화

## 📱 반응형 브레이크포인트

```css
/* 데스크톱 */
@media (min-width: 1024px) { ... }

/* 태블릿 */
@media (max-width: 1024px) { ... }

/* 모바일 */
@media (max-width: 768px) {
  - 네비게이션 메뉴 숨김 (햄버거 메뉴로 대체)
  - 히어로 섹션 패딩 감소
  - 그리드 1열로 변경
}

@media (max-width: 480px) {
  - 폰트 크기 추가 감소
  - 더 작은 아이콘
}
```
